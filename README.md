## 游戏简介

玩家操控一辆像素坦克，在 13×13 的格子地图中消灭敌人、保护自己的基地（鹰）。

游戏包含多个关卡，每一关都有不同的地形、敌人出生点和基地布局。全部敌人被消灭即可进入下一关；基地被摧毁或生命耗尽则游戏结束。

---

## 功能特性

- 双端代码：一套玩法逻辑，分别用 Python/Pygame 与 JavaScript/Canvas 实现
- 像素风画面：坦克、砖墙、钢墙、水域、草地、基地均使用像素矩阵绘制
- 多关卡系统：`levels/` 目录下可自由新增/修改关卡地图
- 敌人 AI：自动寻路、随机转向、定时射击
- 碰撞检测：子弹 vs 墙体、子弹 vs 坦克、坦克 vs 坦克、玩家 vs 敌人接触伤害
- 生命与复活：玩家拥有 3 条命，被击中后在出生点短暂无敌复活
- 首页头像：菜单页显示 `tank_avatar.png` 作为游戏封面/头像

## 快速开始

### 运行 Python 桌面版

1. 安装依赖：

   ```bash
   pip install -r requirements.txt
   ```

2. 启动游戏：

   ```bash
   python main.py
   ```

### 运行微信小游戏版

1. 下载并安装 [微信开发者工具](https://developers.weixin.qq.com/miniprogram/dev/devtools/download.html)。
2. 打开微信开发者工具，选择 **导入项目**。
3. 选择本项目根目录，AppID 可使用测试号或填入项目已有的 `appid`。
4. 点击 **编译** 即可在模拟器或真机预览。

> 提示：微信小游戏内请使用纯 ASCII 文件名引用本地资源，避免中文文件名被 URL 编码后导致文件找不到。

## 关卡编辑

关卡文件位于 `levels/` 目录，每个文件为一个 13 行 × 13 列 的文本地图。

### 地图符号

| 符号 | 含义 |
|---|---|
| `.` | 空地 |
| `#` | 砖墙（可被子弹破坏） |
| `@` | 钢墙（不可破坏） |
| `~` | 水域（坦克不可进入，子弹可飞过） |
| `*` | 草地（坦克可穿过，遮挡视线） |
| `E` | 基地（鹰），必须保护 |
| `P` | 玩家出生点 |
| `1` / `2` / `3` | 敌人出生点 |

### 新增关卡

1. 在 `levels/` 下新建 `level_06.txt`。
2. 按 13×13 格式编辑地图。
3. 游戏会自动按文件名顺序加载关卡（目前需保证文件名连续）。

示例（`level_01.txt`）：

```text
1..........2
............
..##...##...
..##...##...
............
............
...#####....
...#...#....
....###.....
.....P......
.....#......
.....#......
.....E......
```

## 素材说明

- `tank_avatar.png`：首页菜单显示的封面头像，建议使用 **纯 ASCII 文件名**，避免微信开发者工具编码问题。
- `assets/images/`：Python 版首次运行时会自动生成默认像素素材；若删除，下次启动会重新生成。
